# Python e MySQL

Este é um CRUD simples com Python e MySQL para terminal.

Bibliotecas usadas:

* Mysqlclient
* Rich
* Blue
* Isort
* Mypy
* Poetry

## Como rodar esse projeto

```
poetry shell
poetry install
python3.11 base/main.py
```

Como referência, usei o repositório do meu amigo [eddy](https://github.com/eddyyxxyy/crud-python-mysql).
